### Install mongoDB
> brew tap mongodb/brew
> brew update
> brew install mongodb-community@6.0


### Start mongoDB
>  brew services start mongodb/brew/mongodb-community


### Create Express project
> npm init
> npm install express mongoose

### Start the server
> node app.js
